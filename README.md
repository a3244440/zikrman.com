# ZIKRMAN — премиум-каталог одежды

Статический сайт: без сборки, без зависимостей. Работает на любом хостинге, включая Vercel, Netlify и GitHub Pages.

## Структура

- `index.html` — страница
- `styles.css` — стили
- `app.js` — логика (фильтры, карточки, модальное окно, заказ в WhatsApp)
- `products.js` — **все товары и контакты редактируются здесь**

## Как добавить товар

Откройте `products.js` и добавьте блок в массив `PRODUCTS`:

```js
{
  id: 7,
  name: "Название",
  category: "new",          // ключ из CATEGORIES
  price: 25900,             // тенге, только число
  sizes: ["M", "L"],
  image: "https://optim.tildacdn.one/....webp", // прямая ссылка на фото
  description: "Описание изделия."
}
```

Там же замените контакты в `CONTACTS` (номер WhatsApp без «+», ссылка на Instagram).

## Локальный просмотр

Просто откройте `index.html` в браузере — всё работает без сервера.

## Деплой на Vercel через GitHub

1. Создайте репозиторий на GitHub и загрузите эти файлы
   (или через терминал: `git init && git add . && git commit -m "init" && git push`).
2. Зайдите на [vercel.com](https://vercel.com) → **Add New → Project**.
3. Выберите репозиторий → **Import**.
4. Framework Preset: **Other**, ничего не настраивайте → **Deploy**.
5. Через ~30 секунд сайт получит адрес вида `имя-проекта.vercel.app`.
6. Свой домен (zikrman.com): в проекте Vercel → **Settings → Domains** → добавьте домен и пропишите указанные DNS-записи у регистратора.

Каждый новый push в репозиторий автоматически обновляет сайт.
